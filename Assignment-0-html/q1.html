HTML Text Formatting Tags

Tag Description Example
<b> Makes text bold <b>This is bold</b>
<strong> Indicates strong importance (usually renders as bold) <strong>This is strong</strong>
<i> Makes text italic <i>This is italic</i>
<em> Emphasizes text (usually renders as italic) <em>This is emphasized</em>
<u> Underlines the text <u>This is underlined</u>
<s> Strikethrough text (deprecated, use <del> or <strike> instead) <s>This is struck through</s>
<del> Marks deleted text (strikethrough) <del>This is deleted</del>
<ins> Marks inserted text (usually underlined) <ins>This is inserted</ins>
<sub> Subscript text <sub>Subscript</sub>
<sup> Superscript text <sup>Superscript</sup>
<mark> Highlights text <mark>This is highlighted</mark>
<small> Makes text smaller <small>This is small text</small>
<q> Defines a short inline quotation <q>This is a quote</q>
<blockquote> Defines a block quotation <blockquote>This is a blockquote</blockquote>
<abbr> Defines an abbreviation <abbr title="HyperText Markup Language">HTML</abbr>
<cite> Defines a citation <cite>This is a citation</cite>
<code> Defines inline code <code>print('Hello, World!')</code>
<pre> Defines preformatted text <pre> This text preserves whitespace</pre>
<var> Defines a variable in programming or mathematical expressions <var>x</var> = 5;
<time> Defines a specific time or date <time>2024-09-23</time>
